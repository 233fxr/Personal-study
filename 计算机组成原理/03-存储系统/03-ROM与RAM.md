# ROM 与 RAM

## 定义

半导体存储器按存取方式分为两大类：

- **RAM**（Random Access Memory，随机存取存储器）：可读可写，断电后数据丢失（易失性）。CPU 运行时的主存。
- **ROM**（Read-Only Memory，只读存储器）：正常工作时只能读出不能写入，断电后数据不丢失（非易失性）。存放固化的系统程序。

---

## 核心定理 / 性质

### RAM 的分类与对比

| | SRAM（静态RAM） | DRAM（动态RAM） |
|---|---|---|
| 存储单元 | 6 个 MOS 管构成的触发器 | 1 个 MOS 管 + 1 个电容 |
| 破坏性读出 | 否 | 是（读出后需重写） |
| 刷新 | 不需要 | 需要（典型 2ms 内刷新一次） |
| 速度 | 快（ns 级） | 较慢（几十 ns） |
| 集成度 | 低（单元面积大） | 高（单元面积小） |
| 功耗 | 大 | 小 |
| 成本 | 高 | 低 |
| 用途 | Cache | 主存 |

**DRAM 刷新方式**：

| 刷新方式 | 做法 | 特点 |
|---|---|---|
| 集中刷新 | 每隔 2ms 集中一段时间刷新所有行 | 有"死区"，CPU 不能访问 |
| 分散刷新 | 每个存取周期后半段刷新一行 | 无死区，但存取周期变长 |
| 异步刷新 | 2ms 内均匀分散刷新各行 | 折中，最常用 |

### ROM 的分类与演进

| 类型 | 全称 | 写入方式 | 可改写 |
|---|---|---|---|
| MROM | Mask ROM 掩膜 ROM | 厂家在生产时写入 | 不可 |
| PROM | Programmable ROM | 用户一次编程（烧断熔丝） | 不可 |
| EPROM | Erasable PROM | 紫外线照射擦除后重新编程 | 可（需拆下芯片） |
| EEPROM | Electrically EPROM | 电信号擦除和写入 | 可（在线改写） |
| Flash Memory | 闪存 | 电擦除，按块操作 | 可（U盘、SSD） |

**Flash 的两类**：

| | NOR Flash | NAND Flash |
|---|---|---|
| 读取方式 | 按字节随机读，类似 SRAM | 按页读取 |
| 写入/擦除 | 慢 | 快 |
| 用途 | 存放 BIOS、固件代码（XIP 就地执行） | 大容量存储（SSD、U 盘） |

## 计算方法 / 技巧

### 存储容量计算

主存容量 = 存储单元数 × 字长。若用 DRAM 芯片扩展主存，需同时考虑**字扩展**和**位扩展**。

**芯片数** = （目标容量 / 单片容量）×（目标字长 / 单片字长）

**地址线分配**：高位地址经译码器产生片选信号（CS），低位地址同时连到所有芯片的地址引脚上选中片内单元。

### 刷新周期的计算

设 DRAM 有 $r$ 行，刷新周期 $T$（通常 2ms），则：
- 分散刷新下存取周期翻倍，刷新开销 = $r$ 次 / $T$
- 异步刷新间隔 = $T / r$

## 典型例题

**例**：用 256K×4 位的 DRAM 芯片构成 1M×8 位的主存，需多少片？地址线如何分配？

解：字扩展：1M/256K = 4 组；位扩展：8/4 = 2 片/组。共 $4 \times 2 = 8$ 片。

地址线：256K = $2^{18}$，需 18 根低位地址线（A0-A17）连所有芯片选中片内单元。1M/256K = 4，需 2 根高位地址（A18-A19）经 2-4 译码器产生 4 个片选信号，每个选通一组（2 片并行）。

## 易错点

1. ROM 也是随机存取——"随机存取"指可按地址直接访问，不是指可读写。ROM 和 RAM 都是随机存取（区别于顺序存取的磁带）。

2. SRAM 读出非破坏，DRAM 读出是破坏性的——因为 DRAM 靠电容存数据，读出时电容放电，必须在读出后立即重写。

3. DRAM 地址线是分时复用的——行列地址分两次送入（先送行地址（RAS），再送列地址（CAS）），所以 DRAM 的地址引脚数是地址位数的一半。这是常考细节。

4. EPROM 擦除需紫外线照射芯片上的石英窗口（约 20 分钟），EEPROM 可电擦写但次数有限（约 10 万次）。Flash 按块擦除，写前必须先擦。

5. BIOS 早期用 EPROM/EEPROM，现在普遍用 NOR Flash，支持在线更新（刷 BIOS）。
