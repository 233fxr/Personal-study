# 线性微分方程求解（齐次与非齐次）

## 定义

微分方程中**线性**指未知函数 $y$ 及其各阶导数都是一次的、无乘积项（如 $y \cdot y'$ 即为非线性）。考研数学一重点考察一阶线性方程和二阶常系数线性方程，对非线性仅要求可分离变量型和可化为一阶线性的类型。

> **重要区分**：微分方程中"齐次"有**两重完全不同的含义**——
> 1. **线性方程语境**：齐次 = 自由项（右端）为零，即 $y' + P(x)y = 0$ 或 $y'' + py' + qy = 0$
> 2. **可化为齐次的方程语境**：$y' = f(\frac{y}{x})$ 型（通过令 $u = \frac{y}{x}$ 换元化为可分离变量方程）。这与"线性/齐次"的判断无关。

---

## 核心定理 / 性质

### 一阶线性微分方程

**标准形式**：

$$y' + P(x)y = Q(x)$$

- $Q(x) \equiv 0$：**齐次**线性方程
- $Q(x) \not\equiv 0$：**非齐次**线性方程

**求解公式（常数变易法/积分因子法）**：

$$\boxed{y = e^{-\int Pdx}\left(\int Q\,e^{\int Pdx}dx + C\right)}$$

**推导思路**：

1. 先解齐次方程 $y' + P(x)y = 0$：
   $$\frac{dy}{y} = -P(x)dx \;\Longrightarrow\; \ln|y| = -\int Pdx + C_1 \;\Longrightarrow\; y = C e^{-\int Pdx}$$

2. 常数变易：设非齐次解为 $y = C(x) e^{-\int Pdx}$，代入原方程解出 $C(x)$：
   $$C'(x) = Q(x)\,e^{\int Pdx} \;\Longrightarrow\; C(x) = \int Q\,e^{\int Pdx}dx + C$$

3. 代回即得通解公式。

---

### 二阶常系数线性齐次方程

**标准形式**：

$$y'' + p y' + q y = 0 \quad (p,q\text{ 为常数})$$

**解法**：设 $y = e^{rx}$，代入得特征方程

$$r^2 + pr + q = 0$$

根据判别式 $\Delta = p^2 - 4q$：

| $\Delta$ | 特征根 | 通解 |
|---|---|---|
| $\Delta > 0$ | 两不等实根 $r_1 \neq r_2$ | $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| $\Delta = 0$ | 二重实根 $r$ | $y = (C_1 + C_2 x)e^{rx}$ |
| $\Delta < 0$ | 共轭复根 $r = \alpha \pm i\beta$ | $y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$ |

---

### 二阶常系数线性非齐次方程

**标准形式**：

$$y'' + p y' + q y = f(x)$$

**通解结构**：

$$\boxed{y = y_h + y^*}$$ 其中 $y_h$ 为对应齐次方程通解，$y^*$ 为非齐次的一个特解。

**特解 $y^*$ 的设定规则**（待定系数法）：

设 $f(x) = P_m(x)\,e^{\lambda x}$（多项式 $\times$ 指数），或 $f(x) = e^{\alpha x}[P_l(x)\cos\beta x + P_n(x)\sin\beta x]$

| $f(x)$ 的形式 | $y^*$ 的设定 |
|---|---|
| $P_m(x)$ | $y^* = x^k \cdot Q_m(x)$，$Q_m$ 为 $m$ 次待定多项式 |
| $P_m(x)e^{\lambda x}$ | $y^* = x^k \cdot Q_m(x)\,e^{\lambda x}$ |
| $A\cos\beta x + B\sin\beta x$ | $y^* = x^k(M\cos\beta x + N\sin\beta x)$ |
| $e^{\alpha x}(A\cos\beta x + B\sin\beta x)$ | $y^* = x^k \cdot e^{\alpha x}(M\cos\beta x + N\sin\beta x)$ |

其中 $k$ 的取值规则：**$\lambda$（或 $\alpha \pm i\beta$）是特征方程的 $k$ 重根，则前面乘 $x^k$。**

> 不是根 → $k=0$；单根 → $k=1$；二重根 → $k=2$

**多项式次数的确定**：$Q_m(x)$ 的次数取 $f(x)$ 中各部分多项式次数的**最大值**。

---

### n 阶常系数线性方程

上述二阶的方法完全推广到 $n$ 阶：

$$y^{(n)} + a_{n-1}y^{(n-1)} + \cdots + a_1 y' + a_0 y = f(x)$$

特征方程为 $n$ 次多项式 $r^n + a_{n-1}r^{n-1} + \cdots + a_0 = 0$，每个 $m$ 重实根贡献 $m$ 个线性无关解 $(C_0 + C_1 x + \cdots + C_{m-1}x^{m-1})e^{rx}$，每对 $m$ 重共轭复根对应 $m$ 组 $\sin/\cos$ 项。非齐次特解设定规则与二阶一致。

---

### 欧拉方程（Euler Equation）

$$x^2 y'' + a x y' + b y = f(x)$$

**解法**：令 $x = e^t$（$x > 0$）或 $x = -e^t$（$x < 0$），化为常系数线性方程：

$$\frac{d^2y}{dt^2} + (a-1)\frac{dy}{dt} + b y = f(e^t)$$

然后按常系数方法求解，最后将 $t = \ln|x|$ 代回。

---

### 两类非严格线性的可解方程（非线性中的例外）

**可分离变量方程**：

$$\frac{dy}{dx} = g(x)h(y) \;\Longrightarrow\; \int\frac{dy}{h(y)} = \int g(x)dx$$

**伯努利方程**（非线性的 $y^n$ 项出现，但可化为线性）：

$$y' + P(x)y = Q(x)y^n \quad (n \neq 0, 1)$$

令 $z = y^{1-n}$，化为 $\dfrac{dz}{dx} + (1-n)P(x)z = (1-n)Q(x)$，即一阶线性方程。

## 计算方法 / 技巧

### 求解路线图

```
识别方程类型
 ├── 一阶线性？→ 套公式 y = e^{-∫P}(∫Q e^{∫P} + C)
 ├── 可分离变量？→ 分离积分
 ├── 伯努利？→ 换元 z = y^{1-n} → 一阶线性
 ├── 二阶常系数齐次？→ 特征方程 → 三种通解
 ├── 二阶常系数非齐次？→ 齐次通解 + 特解（待定系数法）
 ├── 欧拉方程？→ x = e^t → 常系数
 └── 非上述标准型？→ 检查是否可通过换元化为标准型
```

### 技巧一：非齐次特解"猜"不全时拆项

若 $f(x) = f_1(x) + f_2(x)$，则 $y^* = y_1^* + y_2^*$——分别对 $f_1$ 和 $f_2$ 设特解然后相加（叠加原理）。

### 技巧二：三角函数项的特解

若 $f(x)$ 含 $\sin\beta x$ 或 $\cos\beta x$，即使原式只有 $\sin$ 或只有 $\cos$，**特解中必须同时设 $\sin$ 和 $\cos$ 项**（因为求导会使 $\sin \leftrightarrow \cos$ 互相转换）。

### 技巧三：判断 $k$ 的取值

先写出特征方程，求出特征根。再检查 $f(x)$ 中的 $\lambda$（或 $\alpha \pm i\beta$）：
- 不在特征根中 → $k=0$
- 是单根 → $k=1$
- 是二重根 → $k=2$

## 典型例题

### 例 1（一阶线性非齐次）

求 $y' + \dfrac{2}{x}y = x^3$ 的通解。

**解**：$P(x) = \dfrac{2}{x}$，$Q(x) = x^3$

$$\int Pdx = \int\frac{2}{x}dx = 2\ln|x| = \ln x^2,\quad e^{\int Pdx} = x^2$$

$$y = \frac{1}{x^2}\left(\int x^3 \cdot x^2 dx + C\right) = \frac{1}{x^2}\left(\frac{x^6}{6} + C\right) = \frac{x^4}{6} + \frac{C}{x^2}$$

---

### 例 2（二阶齐次——三种根）

分别求解以下方程的通解：

**(a)** $y'' - 5y' + 6y = 0$（$\Delta > 0$）

$r^2 - 5r + 6 = 0 \;\Longrightarrow\; r_1=2,\; r_2=3$

$y = C_1 e^{2x} + C_2 e^{3x}$

**(b)** $y'' - 4y' + 4y = 0$（$\Delta = 0$）

$r^2 - 4r + 4 = 0 \;\Longrightarrow\; r = 2$（二重）

$y = (C_1 + C_2 x)e^{2x}$

**(c)** $y'' + 2y' + 5y = 0$（$\Delta < 0$）

$r^2 + 2r + 5 = 0 \;\Longrightarrow\; r = -1 \pm 2i$

$y = e^{-x}(C_1\cos 2x + C_2\sin 2x)$

---

### 例 3（非齐次特解设定）

求 $y'' - 3y' + 2y = 3e^{2x}$ 的通解。

**解**：

1. 齐次：$r^2 - 3r + 2 = 0$，$r_1=1$，$r_2=2$，$y_h = C_1 e^x + C_2 e^{2x}$

2. 非齐次：$f(x) = 3e^{2x}$，$\lambda = 2$ 是特征方程的单根 → $k = 1$

   设 $y^* = A x e^{2x}$（**注意：乘了 $x$！**），代入原方程求 $A$：

   $(y^*)' = A e^{2x}(1 + 2x)$，$(y^*)'' = A e^{2x}(4 + 4x)$

   代入：$A e^{2x}(4+4x) - 3A e^{2x}(1+2x) + 2A x e^{2x} = 3e^{2x}$

   系数化简得 $A e^{2x} = 3e^{2x}$，故 $A = 3$

3. $y^* = 3x e^{2x}$

**通解**：$y = C_1 e^x + C_2 e^{2x} + 3x e^{2x}$

---

### 例 4（三角函数非齐次）

求 $y'' + y = \sin x$ 的通解。

**解**：

1. 齐次：$r^2 + 1 = 0$，$r = \pm i$，$y_h = C_1\cos x + C_2\sin x$

2. 非齐次：$f(x) = \sin x$，$\alpha = 0$，$\beta = 1$，而 $\pm i$ 是特征单根 → $k = 1$

   设 $y^* = x(M\cos x + N\sin x)$（**$\sin$ 和 $\cos$ 必须同时设，且乘 $x$**）

   代入求导，比较系数得 $M = -\frac{1}{2}$，$N = 0$

3. $y^* = -\frac{x}{2}\cos x$

**通解**：$y = C_1\cos x + C_2\sin x - \dfrac{x}{2}\cos x$

---

### 例 5（欧拉方程）

求 $x^2 y'' + 3x y' + y = 0$ 的通解。

**解**：令 $x = e^t$（$x>0$），则 $xy' = \frac{dy}{dt}$，$x^2 y'' = \frac{d^2y}{dt^2} - \frac{dy}{dt}$

代入：$\frac{d^2y}{dt^2} + 2\frac{dy}{dt} + y = 0$

特征方程 $r^2 + 2r + 1 = 0$，$r = -1$（二重）

$y(t) = (C_1 + C_2 t)e^{-t}$

代回 $t = \ln x$：$y = \dfrac{C_1 + C_2\ln x}{x}$

## 速查表：非齐次特解设定

| $f(x)$ | 条件的 $k$ | $y^*$ 形式 |
|---|---|---|
| $3x^2 - 2$ | $k=0$（0 不是根） | $ax^2 + bx + c$ |
| $5e^{3x}$ | $\lambda=3$ 不是根→$k=0$ | $A e^{3x}$ |
| $e^{2x}$ | $\lambda=2$ 是单根→$k=1$ | $A\,x\,e^{2x}$ |
| $e^{x}$ | $\lambda=1$ 是二重根→$k=2$ | $A\,x^2 e^{x}$ |
| $\cos 2x$ | $\pm 2i$ 不是根→$k=0$ | $M\cos 2x + N\sin 2x$ |
| $\sin x$ | $\pm i$ 是单根→$k=1$ | $x(M\cos x + N\sin x)$ |

## 易错点

1. **混淆"齐次"两重含义**：线性方程中的齐次指 $Q(x)=0$（$f(x)=0$），而 $y' = f(\frac{y}{x})$ 的叫法是"可化为齐次的方程"，两回事。
2. **二阶齐次复根时漏掉 $\sin$ 项**：通解是 $e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$，不要只写 $\cos$。
3. **非齐次三角函数特解只设了 $\sin$ 或只设了 $\cos$**：必须同时设两项，因为求导会互换。
4. **$\lambda$ 是特征根时忘记乘 $x^k$**：不乘会导致待定系数法无解或解出矛盾方程。
5. **欧拉方程求导不换干净**：$xy' \neq \frac{dy}{dt}$，而是 $x\frac{dy}{dx} = \frac{dy}{dt}$——坐标变换后所有 $x^k y^{(k)}$ 都要用 $\frac{d}{dt}$ 重新表达。
6. **一阶线性公式中 $P(x)$ 的符号**：标准形式是 $y' + P(x)y = Q(x)$，如果原式是 $y' - \frac{2}{x}y = x$，则 $P(x) = -\frac{2}{x}$，不是 $\frac{2}{x}$。
