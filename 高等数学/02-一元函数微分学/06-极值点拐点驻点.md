# 极值点、拐点、驻点

## 定义

三个概念都描述曲线上的特殊点，但判定依据是不同的导数阶数：

| 概念 | 判定依据 | 几何意义 |
|---|---|---|
| 驻点 | $f'(x_0) = 0$ | 切线水平 |
| 极值点 | $f'(x_0) = 0$ 且 $f'$ 在 $x_0$ 两侧变号，或不可导点处 $f'$ 变号 | 局部最大或最小 |
| 拐点 | $f''(x_0) = 0$（或不存在）且 $f''$ 在 $x_0$ 两侧变号 | 凹凸性改变 |

> 逐级递进：一阶导 = 0 → 驻点；一阶导变号 → 极值点；二阶导变号 → 拐点。

---

## 三者关系

```
          ┌──────────────────────────────┐
          │         极值点                │
          │  (f' 变号 或 不可导+f'变号)    │
          │                              │
          │   ┌──────────────────┐       │
          │   │    驻点           │       │
          │   │  (f' = 0)        │       │
          │   │                  │       │
          │   │   ┌────────┐    │       │
          │   │   │ 驻点   │    │       │
          │   │   │ 也是   │    │       │
          │   │   │ 拐点   │    │       │
          │   │   └────────┘    │       │
          │   └──────────────────┘       │
          │                              │
          │   不可导的极值点（如 |x|）     │
          └──────────────────────────────┘

               ┌──────────────────┐
               │    拐点           │
               │  (f'' 变号)       │
               └──────────────────┘
```

### 具体结论

| 命题 | 正误 | 例子 |
|---|---|---|
| 驻点一定是极值点 | ✗ | $y = x^3$ 在 $x=0$：$f'(0)=0$ 但无极值 |
| 极值点一定是驻点 | ✗ | $y = |x|$ 在 $x=0$：不可导但取极小值 |
| 可导的极值点一定是驻点 | ✔ | 费马引理：极值 + 可导 → $f'=0$ |
| 驻点可能是拐点 | ✔ | $y = x^3$ 在 $x=0$：既是驻点又是拐点 |
| 拐点可以是极值点 | ✗ | 极值点处 $f'$ 变号（一阶行为），拐点处 $f''$ 变号（二阶行为），不能同时满足（除了常数函数这种平凡情况） |

---

## 典型例题

**例**：$f(x) = x^4 - 2x^3$，求驻点、极值点、拐点。

解：
$f'(x) = 4x^3 - 6x^2 = 2x^2(2x - 3)$
$f''(x) = 12x^2 - 12x = 12x(x - 1)$

驻点：$f'(x)=0$ 得 $x=0$ 或 $x=3/2$。
- $x=0$：$f'$ 左右均为负（不变号），不是极值点。$f''(0)=0$，$f''$ 在 $x=0$ 两侧由正变负 → 拐点 $(0,0)$。
- $x=3/2$：$f'(3/2)=0$，$f''(3/2)=9>0$ → 极小值点。

拐点：$f''(x)=0$ 得 $x=0$ 或 $x=1$。
- $x=0$：$f''$ 从正变负 → 拐点（同时也是驻点）。
- $x=1$：$f''$ 从负变正 → 拐点 $(1, -1)$。

---

## 易错点

1. **驻点看一阶、极值看一阶变号、拐点看二阶变号**：三个依赖不同阶数，不能串用判断条件。常犯错误是把 $f''(x_0)=0$ 当成极值点的判定条件——那是拐点的判定条件。

2. **不可导点必须单独检验**：$f'(x_0)$ 不存在不能直接跳过。如 $y=|x|$ 在 $x=0$ 是不可导的极小值点，$y=x^{1/3}$ 在 $x=0$ 处导数为无穷（切线垂直），也是拐点（二阶导变号）。

3. **费马引理的逆不成立**：$f'(x_0)=0$ 是极值的必要条件（可导时），不是充分条件。$x^3$ 在 0 处一次导为零但无极值。

4. **极值点和拐点不可能重合（除非常数）**：极值局部单调→不单调→单调，拐点凹→不凹→凹。一个点如果同时是极值点和拐点，则 $f'$ 和 $f''$ 都在此处变号——这要求 $f$ 在局部不是解析的，考试中几乎不可能出现（理论上是存在的，如分段定义的病态函数）。
