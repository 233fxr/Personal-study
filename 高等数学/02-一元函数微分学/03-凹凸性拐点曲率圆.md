# 凹凸性、拐点、曲率圆

## 定义

**凹凸性**：设 $f$ 在区间 $I$ 上二阶可导。
- 若 $f''(x) > 0$，则 $f$ 在 $I$ 上是**凹的**（下凸，开口向上，$\smile$）
- 若 $f''(x) < 0$，则 $f$ 在 $I$ 上是**凸的**（上凸，开口向下，$\frown$）

**拐点**：曲线凹凸性发生改变的点。即 $f''$ 变号的点。

> 注意考研教材用词：同济版"凹"=下凸=开口向上= $f''>0$，"凸"=上凸=开口向下= $f''<0$。不同教材用语可能相反，考试以 $f''$ 的正负为准。

**曲率**：曲线弯曲程度的度量，定义为切线方向角对弧长的变化率：

$$\kappa = \left| \frac{y''}{(1+y'^2)^{3/2}} \right|$$

**曲率半径**：$\rho = \dfrac{1}{\kappa}$

**曲率圆（密切圆）**：在点 $M(x_0, y_0)$ 处与曲线有相同切线、相同曲率的圆。圆心在法线上，距 $M$ 的距离为曲率半径 $\rho$。

---

## 核心定理 / 性质

### 拐点的判定

| 条件 | 结论 |
|---|---|
| $f''(x_0) = 0$ 且 $f''$ 在 $x_0$ 两侧变号 | $(x_0, f(x_0))$ 是拐点 |
| $f''(x_0) = 0$，$f'''(x_0) \neq 0$ | $(x_0, f(x_0))$ 是拐点（第三阶导数判别法，更快捷） |
| $f''(x_0)$ 不存在 | 也可能是拐点（只要 $f''$ 在该点两侧变号） |

### 曲率圆的几何定位

给定曲线 $y = f(x)$ 在点 $M(x_0, y_0)$ 处：

- 曲率中心 $(\alpha, \beta)$：
  $$\alpha = x_0 - \frac{y'(1+y'^2)}{y''}, \quad \beta = y_0 + \frac{1+y'^2}{y''}$$

- 曲率圆方程：$(x - \alpha)^2 + (y - \beta)^2 = \rho^2$

> 曲率中心在法线上，位于曲线的凹侧。$y''>0$ 时在曲线上方，$y''<0$ 时在曲线下方。

---

## 计算方法 / 技巧

### 凹凸区间和拐点的求解步骤

1. 求 $f''(x)$
2. 找出 $f''(x) = 0$ 的点和 $f''(x)$ 不存在的点
3. 用这些点划分定义域，列表判断各区间 $f''$ 的符号
4. $f''$ 变号的点即为拐点

### 曲率圆的实用计算

1. 求 $y'$ 和 $y''$ 在 $x_0$ 处的值
2. 代入曲率公式得 $\kappa$ 和 $\rho$
3. 法线斜率 $k_\perp = -1/y'$（若 $y'\neq 0$）
4. 沿法线凹侧方向走 $\rho$ 得圆心

## 典型例题

**例1**：求 $y = x^3 - 3x$ 的凹凸区间和拐点。

解：$y' = 3x^2 - 3$，$y'' = 6x$。
- $x < 0$：$y'' < 0$，凸区间
- $x = 0$：$y'' = 0$，且 $y''$ 由负变正，拐点为 $(0, 0)$
- $x > 0$：$y'' > 0$，凹区间

**例2**：求 $y = x^3$ 在 $(1, 1)$ 处的曲率圆。

解：$y' = 3x^2$，在 $x=1$ 处 $y'=3$。$y''=6x$，在 $x=1$ 处 $y''=6$。

曲率 $\kappa = \left| \frac{6}{(1+9)^{3/2}} \right| = \frac{6}{10\sqrt{10}}$，$\rho = \frac{10\sqrt{10}}{6}$。

曲率中心：$\alpha = 1 - \frac{3(1+9)}{6} = 1-5 = -4$，$\beta = 1 + \frac{10}{6} = \frac{8}{3}$。

曲率圆：$(x+4)^2 + (y-\frac{8}{3})^2 = \frac{250}{9}$。

## 易错点

1. **$f''(x_0) = 0$ 不保证是拐点**：如 $y = x^4$，$y''(0)=0$，但 $y''=12x^2 \geq 0$ 不变号，$(0,0)$ 不是拐点。必须检验 $f''$ 是否变号或 $f'''\neq 0$。

2. **拐点必须是曲线上的点**：拐点是有序对 $(x_0, f(x_0))$，不能只写 $x_0$。

3. **极值点和拐点的混淆**：极值点看 $f'$ 的符号变化，拐点看 $f''$ 的符号变化。驻点 $(f'=0)$ 可能是极值点，$f''=0$ 的点可能是拐点，两者完全独立——$y=x^3$ 在 $x=0$ 处同时是驻点和拐点，但这是巧合。

4. **曲率公式分子带绝对值，曲率中心不带**：曲率 $\kappa \geq 0$ 总是非负的；但曲率中心公式中 $y''$ 不带绝对值——符号决定了圆心在法线的哪一侧（凹侧）。

5. **直线和拐点**：直线上 $y'' \equiv 0$，每一点的曲率为零，曲率半径无穷大。直线上的点都不是拐点（凹凸性没有"发生改变"）。
