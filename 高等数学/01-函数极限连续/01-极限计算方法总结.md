# 极限计算方法总结：等价替换、泰勒展开与七种未定式

## 定义

极限计算是考研数学一的基础，核心目标是在 $x \to x_0$（或 $x \to \infty$）时求 $f(x)$ 的极限值。常用工具包括等价无穷小替换、泰勒展开、洛必达法则，以及针对不同未定式的转化策略。

---

## 核心定理 / 性质

### 1. 等价无穷小替换（$x \to 0$）

> **核心原则**：等价无穷小只能替换**乘除因子**，不能在加减中直接替换，除非泰勒展开到足够高阶后确认可以。

**基本替换（一阶）**：

| 无穷小 | 等价量 | | 无穷小 | 等价量 |
|---|---|---|---|---|
| $\sin x$ | $x$ | | $\tan x$ | $x$ |
| $\arcsin x$ | $x$ | | $\arctan x$ | $x$ |
| $\ln(1+x)$ | $x$ | | $e^x - 1$ | $x$ |
| $a^x - 1$ | $x \ln a$ | | $(1+x)^\alpha - 1$ | $\alpha x$ |
| $1 - \cos x$ | $\frac{1}{2}x^2$ | | | |

**做差替换（用泰勒，不可直接用等价无穷小相减）**：

| 差式 | 等价量 | 阶 |
|---|---|---|
| $x - \sin x$ | $\frac{1}{6}x^3$ | 3 |
| $\tan x - x$ | $\frac{1}{3}x^3$ | 3 |
| $\arcsin x - x$ | $\frac{1}{6}x^3$ | 3 |
| $x - \arctan x$ | $\frac{1}{3}x^3$ | 3 |
| $\tan x - \sin x$ | $\frac{1}{2}x^3$ | 3 |

### 2. 泰勒展开（Maclaurin，$x \to 0$）

常用展开式（记熟到 $x^3$ 阶，做差问题至少展到 $x^3$）：

- $e^x = 1 + x + \dfrac{x^2}{2!} + \dfrac{x^3}{3!} + o(x^3)$
- $\sin x = x - \dfrac{x^3}{3!} + \dfrac{x^5}{5!} + o(x^5)$
- $\cos x = 1 - \dfrac{x^2}{2!} + \dfrac{x^4}{4!} + o(x^4)$
- $\ln(1+x) = x - \dfrac{x^2}{2} + \dfrac{x^3}{3} + o(x^3)$
- $(1+x)^\alpha = 1 + \alpha x + \dfrac{\alpha(\alpha-1)}{2!}x^2 + o(x^2)$
- $\arctan x = x - \dfrac{x^3}{3} + \dfrac{x^5}{5} + o(x^5)$
- $\arcsin x = x + \dfrac{x^3}{6} + o(x^3)$
- $\tan x = x + \dfrac{x^3}{3} + o(x^3)$

> **泰勒做差思路**：当两个函数等价主项相同（如同一阶项皆是 $x$），做差后主项消去，需展开到下一阶得差式等价量。如 $\sin x - x = -\dfrac{x^3}{6} + o(x^3)$，故 $\sin x - x \sim -\dfrac{x^3}{6}$。

### 3. 洛必达法则

若 $\displaystyle\lim \dfrac{f(x)}{g(x)}$ 为 $\dfrac{0}{0}$ 或 $\dfrac{\infty}{\infty}$ 型，且 $\displaystyle\lim \dfrac{f'(x)}{g'(x)}$ 存在（或为无穷大），则

$$\lim \frac{f(x)}{g(x)} = \lim \frac{f'(x)}{g'(x)}$$

> **注意**：洛必达法则仅适用于 $0/0$ 和 $\infty/\infty$，其他未定式必须先转化为这两者。此外，有些情况下求导后反而更复杂（如含 $\sin x$、$e^x$ 等周期或指数项），此时等价替换 + 泰勒往往更高效。

## 七种未定式详解

### ① $\dfrac{0}{0}$ 型

**策略（按优先顺序）**：

1. **因式分解约分**：若分子分母有公因式（尤其多项式），先约分
2. **等价无穷小替换**：在乘除因子中替换，降低复杂度
3. **泰勒展开**：将分子分母各展开到同阶，比较最低阶项系数
4. **洛必达法则**：多次使用直到出结果（注意检查条件）
5. **根式有理化**：若含 $\sqrt{a} - \sqrt{b}$ 形式，乘共轭式化简

**例**：求 $\displaystyle\lim_{x \to 0} \frac{\tan x - \sin x}{x^3}$

解：$\tan x = x + \frac{x^3}{3} + o(x^3)$，$\sin x = x - \frac{x^3}{6} + o(x^3)$
做差得 $\frac{x^3}{2} + o(x^3)$，故极限为 $\dfrac{1}{2}$。

（另法：$\tan x - \sin x = \tan x(1-\cos x) \sim x \cdot \frac{x^2}{2} = \frac{x^3}{2}$）

---

### ② $\dfrac{\infty}{\infty}$ 型

**策略**：

1. **分子分母同除以最高次幂**：适用于多项式或有理函数，直接比次数
2. **洛必达法则**：多项式或可微函数的 $\infty/\infty$
3. **抓大放小（增长速率比较）**：当 $x \to \infty$ 时，指数函数 >> 幂函数 >> 对数函数
   - 例如 $\displaystyle\lim_{x \to \infty} \frac{x^{100}}{e^x} = 0$（指数增长碾压幂函数）
   - $\displaystyle\lim_{x \to \infty} \frac{\ln x}{x^p} = 0$（$p > 0$）

**例**：求 $\displaystyle\lim_{x \to \infty} \frac{3x^3 + 2x^2 - 5}{x^3 - 7x + 1}$

解：分子分母同除以 $x^3$，得 $\displaystyle\lim_{x \to \infty} \frac{3 + \frac{2}{x} - \frac{5}{x^3}}{1 - \frac{7}{x^2} + \frac{1}{x^3}} = 3$

---

### ③ $0 \cdot \infty$ 型

**策略**：**转化为 $\dfrac{0}{0}$ 或 $\dfrac{\infty}{\infty}$**

- 写法一：$0 \cdot \infty = \dfrac{0}{\ 1/\infty\ } = \dfrac{0}{0}$（把无穷大放分母）
- 写法二：$0 \cdot \infty = \dfrac{\infty}{\ 1/0\ } = \dfrac{\infty}{\infty}$（把无穷小放分母）

**选哪边？** 看转化后谁的导数更简单，优先把简单函数放分子、把复杂的放分母。

**例**：求 $\displaystyle\lim_{x \to 0^+} x \ln x$

解：化为 $\dfrac{0}{0}$ 型：$\displaystyle\lim_{x \to 0^+} \frac{\ln x}{1/x}$

洛必达：$\displaystyle\lim_{x \to 0^+} \frac{1/x}{-1/x^2} = \lim_{x \to 0^+} (-x) = 0$

---

### ④ $\infty - \infty$ 型

**策略**：

1. **分式差**：通分合并为一个分式 $\to$ $0/0$ 或 $\infty/\infty$
2. **根式差**（$\sqrt{a} - \sqrt{b}$）：乘共轭式有理化
3. **提取公因子**：提出增长速度最快的项，化为确定形式
4. **泰勒展开**：将各项展开后做差

**例 1（分式差）**：求 $\displaystyle\lim_{x \to 0} \left( \frac{1}{x} - \frac{1}{\sin x} \right)$

解：通分 $\to \dfrac{\sin x - x}{x\sin x}$，由泰勒 $\sin x - x \sim -\dfrac{x^3}{6}$，$x\sin x \sim x^2$
故 $\sim -\dfrac{x}{6} \to 0$

**例 2（根式差）**：求 $\displaystyle\lim_{x \to +\infty} \big( \sqrt{x^2+x} - x \big)$

解：有理化 $\dfrac{(x^2+x) - x^2}{\sqrt{x^2+x} + x} = \dfrac{x}{\sqrt{x^2+x} + x} = \dfrac{1}{\sqrt{1 + \frac{1}{x}} + 1} \to \dfrac{1}{2}$

---

### ⑤ $0^0$ 型

**策略**：取对数转化

1. 设 $y = f(x)^{g(x)}$，则 $\ln y = g(x) \cdot \ln f(x)$
2. 当 $f(x) \to 0^+$ 时 $\ln f(x) \to -\infty$，$g(x) \to 0$
3. 于是 $\ln y$ 为 $0 \cdot (-\infty)$ 型 → 按 ③ 处理
4. 求得 $\displaystyle\lim \ln y = A$ 后，$\displaystyle\lim y = e^A$

**例**：求 $\displaystyle\lim_{x \to 0^+} x^x$

解：令 $y = x^x$，$\ln y = x \ln x$

由上例已知 $\displaystyle\lim_{x \to 0^+} x \ln x = 0$，故 $\displaystyle\lim_{x \to 0^+} x^x = e^0 = 1$

---

### ⑥ $\infty^0$ 型

**策略**：同样取对数转化

1. 设 $y = f(x)^{g(x)}$，$\ln y = g(x) \cdot \ln f(x)$
2. 当 $f(x) \to \infty$，$g(x) \to 0$ 时，$\ln y$ 为 $0 \cdot \infty$ 型
3. 按 ③ 处理，求得 $\lim \ln y$ 后指数还原

**例**：求 $\displaystyle\lim_{x \to +\infty} x^{1/x}$

解：令 $y = x^{1/x}$，$\ln y = \dfrac{\ln x}{x}$，此为 $\infty/\infty$ 型

洛必达：$\displaystyle\lim_{x \to +\infty} \frac{1/x}{1} = 0$

故 $\displaystyle\lim_{x \to +\infty} x^{1/x} = e^0 = 1$

---

### ⑦ $1^\infty$ 型（考研最高频）

**核心公式**：设 $\displaystyle\lim f(x) = 1$，$\displaystyle\lim g(x) = \infty$，且 $\displaystyle\lim g(x)\big(f(x)-1\big) = A$ 存在，则：

$$\boxed{\lim \big[f(x)\big]^{g(x)} = e^{\displaystyle\lim g(x)(f(x)-1)} = e^A}$$

**等价形式**：当 $\alpha \to 0$，$\beta \to \infty$ 时

$$\lim (1 + \alpha)^\beta = e^{\displaystyle\lim \alpha\beta}$$

**三种常用解法**：

| 方法 | 适用场景 |
|---|---|
| 公式法 $e^{\lim g(f-1)}$ | 能直接算出 $\lim g(f-1)$ |
| 重要极限 $\lim(1+\frac{1}{n})^n = e$ | 能凑出 $(1+\frac{1}{t})^t$ 形式 |
| 取对数 + 洛必达 | 公式法失效或更复杂时 |

**例 1（公式法）**：求 $\displaystyle\lim_{x \to 0} (1 + 2x)^{\frac{1}{\sin x}}$

解：$f=1+2x \to 1$，$g=\frac{1}{\sin x} \to \infty$

$g(f-1) = \dfrac{2x}{\sin x} \sim \dfrac{2x}{x} = 2$，故极限 $= e^2$

**例 2（凑重要极限）**：求 $\displaystyle\lim_{n \to \infty} \left(1 + \frac{1}{n}\right)^{2n}$

解：$= \left[\left(1 + \frac{1}{n}\right)^n\right]^2 \to e^2$

## 七种未定式速查表

| # | 类型 | 核心转化 | 首选方法 |
|---|---|---|---|
| ① | $\dfrac{0}{0}$ | 无需转化 | 等价替换 → 泰勒 → 洛必达 |
| ② | $\dfrac{\infty}{\infty}$ | 无需转化 | 除以最高次幂 → 洛必达 |
| ③ | $0 \cdot \infty$ | → $\dfrac{0}{0}$ 或 $\dfrac{\infty}{\infty}$ | 把简单的放分子 |
| ④ | $\infty - \infty$ | 通分 / 有理化 | → 化为 ① 或 ② |
| ⑤ | $0^0$ | 取对数 → $0 \cdot (-\infty)$ | → ③ → ①/② → 指数还原 |
| ⑥ | $\infty^0$ | 取对数 → $0 \cdot \infty$ | → ③ → ①/② → 指数还原 |
| ⑦ | $1^\infty$ | 公式 $e^{\lim g(f-1)}$ | 直接套公式（优先） |

## 易错点

1. **加减中误用等价无穷小**：$\tan x - \sin x$ 中 $\tan x \sim x$，$\sin x \sim x$，但 $\tan x - \sin x \not\sim 0$！必须用泰勒或提因子。
2. **洛必达滥用**：有些 $0/0$ 型求导后更复杂，等价替换 + 泰勒往往更快。
3. **$1^\infty$ 忘指数还原**：算出 $\lim v(u-1) = A$ 后，答案必须是 $e^A$，不是 $A$。
4. **无穷小阶数不够**：做差时展开阶数必须超过两者差的主项阶数。如 $\sin x$ 和 $x$ 做差，一阶不够，需展到三阶。
5. **等价替换忽略条件**：$x$ 必须是趋向于 $0$ 的量。若 $x \to a \neq 0$，需先变量代换 $t = x-a \to 0$。
6. **$0 \cdot \infty$ 选错转化方向**：把复杂函数放分母可能导致洛必达后更复杂，应优先把简单函数放分子。
7. **$\infty - \infty$ 不转化为分式就用等价**：必须先转化为一个整体（通分或有理化）再做极限。
