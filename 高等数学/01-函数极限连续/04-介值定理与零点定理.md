# 介值定理与零点定理

## 定义

### 介值定理（Intermediate Value Theorem，IVT）

若函数 $f(x)$ 在闭区间 $[a,b]$ 上**连续**，且 $f(a) \neq f(b)$，则对于任意介于 $f(a)$ 和 $f(b)$ 之间的值 $\eta$，**至少存在一点** $\xi \in (a,b)$，使得 $f(\xi) = \eta$。

通俗地说：**连续函数在区间上不会跳过任何中间值**——函数图像是从 $f(a)$ 到 $f(b)$ 的一条连续曲线，途中必须经过两者之间的每一个值。

### 零点定理（Brouwer 不动点定理的特例，也称根的存在定理）

若 $f(x)$ 在 $[a,b]$ 上连续，且 $f(a) \cdot f(b) < 0$（即两端点**异号**），则 $\exists\, \xi \in (a,b)$，使得 $f(\xi) = 0$。

零点定理是介值定理在 $\eta = 0$ 时的直接推论。

---

## 核心定理 / 性质

### 1. 介值定理的严格证明（基于确界定理）

**前提**：实数完备性——非空有上界的数集必有上确界。

**证明**：

设 $f$ 在 $[a,b]$ 上连续，令 $c$ 介于 $f(a)$ 和 $f(b)$ 之间。

构造集合 $S = \{x \in [a,b] \mid f(x) \leq c\}$。

- $S$ 非空（$a \in S$，因为 $f(a) \leq c$ 或 $f(a) \geq c$，取对应的方向）
- $S$ 有上界 $b$

令 $\xi = \sup S$。

由 $f$ 的连续性：
- 若 $f(\xi) > c$，则 $\xi$ 附近 $f(x) > c$，$\xi$ 不是上确界（矛盾）
- 若 $f(\xi) < c$，则 $\xi$ 附近 $f(x) < c$，存在比 $\xi$ 更大的元素在 $S$ 中（矛盾）

故必有 $f(\xi) = c$。∎

> **考研要求**：不要求证明介值定理本身（属于实数完备性理论），但要求会用介值定理和零点定理解题。

### 2. 介值定理的直观理解

```
        f(b)
         │
         ├─── 水平线 y = η
         │     ↓
        f(a)   ξ (介值点)
         │
         └─────────
         a    ξ    b
```

闭区间上连续 ⇒ 值域也是闭区间：$f([a,b]) = [m, M]$，其中 $m = \min f$，$M = \max f$。

介值定理等价于：连续函数的像集是区间。

### 3. 推广形式

| 形式 | 内容 |
|---|---|
| 标准介值定理 | $f$ 在 $[a,b]$ 连续 ⇒ 取遍 $f(a)$ 和 $f(b)$ 之间的一切值 |
| **零点定理** | $f(a)f(b) < 0$ ⇒ $\exists\, f(\xi) = 0$ |
| 介值定理的推论 | $f$ 在 $[a,b]$ 连续 ⇒ 值域为 $[m, M]$ |
| **不动点定理** | $f$ 在 $[a,b]$ 连续，$f([a,b]) \subseteq [a,b]$ ⇒ $\exists\, \xi$ 使 $f(\xi)=\xi$ |

### 4. 介值定理的逆否命题

> 若 $f(a)$ 和 $f(b)$ 之间有一个值 $\eta$ 使得 $f(x) \neq \eta$ 对所有 $x \in [a,b]$ 成立，则 $f$ 在 $[a,b]$ 上不连续。

这等价于：连续函数不可能跳过任何中间值。

## 计算方法 / 技巧

### 应用模式一：证明方程有根

**核心步骤**：令 $F(x) = f(x) - g(x)$（或适当变形）→ 找两点使 $F$ 异号 → 零点定理。

### 应用模式二：证明函数取到特定值

将问题转化为 $F(x) = f(x) - c$ 的零点问题。

### 应用模式三：介值定理 + 单调性确定根的个数

- 介值定理保证**至少一个根**
- 单调性（导数恒正/恒负）保证**至多一个根**
- 两者结合 = **恰有一个根**

### 应用模式四：不动点问题

构造 $F(x) = f(x) - x$，若 $F(a) \geq 0$ 且 $F(b) \leq 0$，由零点定理 $\exists\, \xi$ 使 $f(\xi) = \xi$。

### 介值定理 vs 中值定理的区别

| | 介值定理 | 中值定理（拉格朗日） |
|---|---|---|
| 条件 | 仅需**连续** | 需连续 + **可导** |
| 结论 | 函数值存在 | 导数存在 |
| 本质 | 连续函数的性质 | 可导函数的性质 |
| 证明工具 | 实数完备性（确界） | 罗尔定理 |

两者名字都带"中值"但完全不同——**介值定理是关于函数值的，中值定理是关于导数值的**。

## 典型例题

### 例 1（零点定理基础）

证明方程 $x^5 - 3x + 1 = 0$ 在区间 $(0,1)$ 内至少有一个根。

**证**：令 $f(x) = x^5 - 3x + 1$。
$f(0) = 1 > 0$，$f(1) = 1 - 3 + 1 = -1 < 0$。
$f(0) \cdot f(1) < 0$，且 $f$ 在 $[0,1]$ 上连续（多项式函数）。
由**零点定理**，$\exists\, \xi \in (0,1)$ 使 $f(\xi) = 0$。∎

---

### 例 2（介值定理应用）

设 $f(x)$ 在 $[0,1]$ 上连续，$f(0)=0$，$f(1)=2$。证明 $\exists\, \xi \in (0,1)$ 使 $f(\xi) = 1$。

**证**：$f(0)=0 < 1 < 2 = f(1)$，1 介于 $f(0)$ 和 $f(1)$ 之间。
由**介值定理**，$\exists\, \xi \in (0,1)$ 使 $f(\xi) = 1$。∎

---

### 例 3（不动点问题）

设 $f(x)$ 在 $[0,1]$ 上连续，且 $0 \leq f(x) \leq 1$。证明 $\exists\, \xi \in [0,1]$ 使 $f(\xi) = \xi$。

**证**：令 $F(x) = f(x) - x$。
$F(0) = f(0) - 0 = f(0) \geq 0$。
$F(1) = f(1) - 1 \leq 0$（因为 $f(1) \leq 1$）。

若 $F(0)=0$ 或 $F(1)=0$，则 $\xi=0$ 或 $\xi=1$ 为所求。
若 $F(0)>0$ 且 $F(1)<0$，则 $F(0) \cdot F(1) < 0$，由**零点定理** $\exists\, \xi \in (0,1)$ 使 $F(\xi)=0$，即 $f(\xi)=\xi$。∎

---

### 例 4（方程根的存在性 + 单调性定唯一）

证明方程 $x^3 - 3x + 1 = 0$ 在区间 $(1,2)$ 内有唯一实根。

**证**：

令 $f(x) = x^3 - 3x + 1$。

**存在性**：$f(1) = 1-3+1 = -1 < 0$，$f(2)=8-6+1=3>0$。
由零点定理，$\exists\, \xi \in (1,2)$ 使 $f(\xi)=0$。

**唯一性**：$f'(x) = 3x^2 - 3 = 3(x^2-1)$。
当 $x \in (1,2)$ 时 $f'(x) > 0$，$f$ 在 $[1,2]$ 上严格单调递增。
故 $f$ 在 $(1,2)$ 内至多有一个零点。

综合存在性和唯一性，方程在 $(1,2)$ 内有唯一实根。∎

---

### 例 5（介值定理 + 周期性综合）

设 $f(x)$ 在 $(-\infty,+\infty)$ 上连续，且 $f(x + T) = f(x)$（$T$ 为周期）。若 $f(0) \neq f(T/2)$，证明方程 $f(x) = f(x + T/2)$ 必有实根。

**证**：

令 $F(x) = f(x) - f(x + T/2)$。

$F(0) = f(0) - f(T/2)$，设其符号为正（为负同理）。
$F(T/2) = f(T/2) - f(T + T/2) = f(T/2) - f(T/2) = 0$？不对，注意：
$F(T/2) = f(T/2) - f(T) = f(T/2) - f(0)$（由周期性 $f(T)=f(0)$）。

所以 $F(T/2) = f(T/2) - f(0) = -[f(0)-f(T/2)] = -F(0)$。

由于 $f(0) \neq f(T/2)$，$F(0) \neq 0$，且 $F(0)$ 与 $F(T/2)$ 异号。

由零点定理，$\exists\, \xi \in (0, T/2)$ 使 $F(\xi) = 0$，即 $f(\xi) = f(\xi + T/2)$。∎

---

### 例 6（介值定理证明等分问题）

设 $f(x)$ 在 $[0,1]$ 上连续，$f(0)=f(1)$。证明 $\exists\, \xi \in [0,1]$ 使 $f(\xi) = f(\xi + \frac12)$。

**证**：

令 $F(x) = f(x) - f(x + \frac12)$，定义域为 $[0, \frac12]$。

$F(0) = f(0) - f(\frac12)$。
$F(\frac12) = f(\frac12) - f(1) = f(\frac12) - f(0) = -F(0)$。

若 $F(0) = 0$，则 $\xi = 0$ 为所求。
若 $F(0) \neq 0$，则 $F(0)$ 与 $F(\frac12)$ 异号，由零点定理 $\exists\, \xi \in (0, \frac12)$ 使 $F(\xi)=0$，即 $f(\xi)=f(\xi+\frac12)$。

> 这个结论可以推广：若 $f$ 在 $[0,1]$ 上连续且 $f(0)=f(1)$，则对任意正整数 $n$，存在两点相距 $1/n$ 的函数值相等。

---

### 例 7（连续函数的像集）

设 $f(x)$ 在 $[0,2]$ 上连续，$f(0)=1$，$f(1)=2$，$f(2)=0$。求 $f$ 的值域的一个下界判断。

**分析**：由于 $f(2)=0$ 且 $f(1)=2$，在 $[1,2]$ 上由介值定理，$f$ 取遍 $0$ 到 $2$ 之间所有值。但由于不知 $f$ 在 $[0,1]$ 上的单调性，无法确定最小值是否小于 $0$。只知道值域包含 $[0,2]$ 且包含 $1$（在 $[0,1]$ 上介于 $1$ 和 $2$ 之间）。

> **启发**：介值定理保证取遍"区间内部"的值，但不保证区间端点是最值。

## 易错点

1. **介值定理 ≠ 中值定理**：名字相似但完全不同。介值定理（IVT）只需连续，中值定理（MVT）需要可导。"介"是"介于中间"，"中值"是区间内部某点的导数值
2. **零点定理需要两端异号**：$f(a)f(b) < 0$ 不能写成 $f(a) < 0 < f(b)$——后者只是异号的一种特殊情况，但正确。注意 $a < b$ 不能反过来
3. **零点定理只保证存在性，不保证唯一性**：如果还需要唯一，必须额外证明单调性
4. **介值定理要求闭区间连续**：开区间连续或闭区间有间断点时定理不成立。如 $f(x)=\frac{1}{x}$ 在 $(0,1)$ 上不取无界值
5. **零点定理中的零点 $\xi$ 不一定唯一**（已如第 3 点）
6. **不动点构造 $F(x)=f(x)-x$ 时，$\xi$ 可能在端点**：记得检查 $F(a)=0$ 或 $F(b)=0$ 的情况

## 介值定理与其他定理的关联

```
        确界定理（实数完备性）
              ↓
        介值定理 (IVT)
        /           \
   零点定理       介值定理的一般形式
   （根的存在）    （连续函数像集为区间）
        ↓               ↓
   方程求根       不动点定理
   + 单调性       + 压缩映射
   → 唯一根       → 迭代收敛
        ↓               ↓
    微分方程         数值分析
    解的存在性       方程的近似解
```

### 关联知识

- 与「中值定理」名字易混，但本质不同——介值定理属于**连续函数性质**（拓扑性），中值定理属于**可导函数性质**（微分性）
- 与「极限计算方法」同属函数极限连续章——极限解决局部行为，介值定理解决区间上的整体行为
- 零点定理是证明方程有根的**首要工具**，配合单调性可确定唯一根
