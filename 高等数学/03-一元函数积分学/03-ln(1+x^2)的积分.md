# ln(1+x²) 的积分及变形

## 定义

$\displaystyle\int \ln(1+x^2)dx$ 是对数函数与多项式复合的积分，核心方法是**分部积分 $\to$ 有理函数拆分**。它是考研中"分部积分 + 多项式除法/拆项"的经典母题，由此可衍生大量变体。

---

## 核心定理 / 性质

### 不定积分

$$\boxed{\int \ln(1+x^2)dx = x\ln(1+x^2) - 2x + 2\arctan x + C}$$

**推导**：分部积分，$u = \ln(1+x^2)$，$dv = dx$，则 $du = \dfrac{2x}{1+x^2}dx$，$v = x$

$$\int \ln(1+x^2)dx = x\ln(1+x^2) - \int x\cdot\frac{2x}{1+x^2}dx = x\ln(1+x^2) - 2\int\frac{x^2}{1+x^2}dx$$

拆 $\dfrac{x^2}{1+x^2} = 1 - \dfrac{1}{1+x^2}$，代入：

$$= x\ln(1+x^2) - 2x + 2\arctan x + C$$

### 常用定积分

| 积分 | 结果 |
|---|---|
| $\displaystyle\int_0^1 \ln(1+x^2)dx$ | $\ln 2 - 2 + \dfrac{\pi}{2}$ |
| $\displaystyle\int_{-a}^{a} \ln(1+x^2)dx$ | $2\big[a\ln(1+a^2) - 2a + 2\arctan a\big]$（偶函数对称） |

## 计算方法 / 技巧

### 技巧一：对数积分一律先分部积分

对 $\displaystyle\int \ln(f(x))dx$ 型，令 $u = \ln(f(x))$，$dv = dx$：

$$\int \ln(f(x))dx = x\ln(f(x)) - \int\frac{x f'(x)}{f(x)}dx$$

然后对被积有理函数做拆分或换元。

### 技巧二：假分式拆项

$\dfrac{x^2}{1+x^2}$ 是假分式（分子次数 = 分母次数），拆为：

$$\frac{x^2}{1+x^2} = 1 - \frac{1}{1+x^2}$$

一般公式（$n \ge m$ 时）：

$$\frac{x^n}{a^2 + x^2} = x^{n-2} - \frac{a^2 x^{n-2}}{a^2 + x^2}$$

### 技巧三：$\ln$ 与 $\arctan$ 的固定搭配

$\displaystyle\int \frac{dx}{1+x^2} = \arctan x + C$，这是 $\ln(1+x^2)$ 积分中 $\arctan$ 反复出现的原因——分部后产生了 $\dfrac{1}{1+x^2}$。

### 常见变形

**变形一**：$\displaystyle\int \ln(a^2+x^2)dx$
$$= x\ln(a^2+x^2) - 2x + 2a\arctan\frac{x}{a} + C$$

**变形二**：$\displaystyle\int x\ln(1+x^2)dx$ — 凑微分更快
$$= \frac{1}{2}\int \ln(1+x^2)d(1+x^2) = \frac{1+x^2}{2}\ln(1+x^2) - \frac{1+x^2}{2} + C$$

**变形三**：$\displaystyle\int x^2\ln(1+x^2)dx$ — 分部后拆 $\dfrac{x^4}{1+x^2}$
$$= \frac{x^3}{3}\ln(1+x^2) - \frac{2}{3}\Big(\frac{x^3}{3} - x + \arctan x\Big) + C$$

**变形四**：$\displaystyle\int \frac{\ln(1+x^2)}{x}dx$ — ⚠ **非初等积分**（dilogarithm），考研不要求初等表达式。见到绕行。

## 典型例题

### 例 1（基本型）

求 $\displaystyle\int \ln(1+x^2)dx$

**解**：分部 $\to$ $\dfrac{x^2}{1+x^2}$ 拆为 $1 - \dfrac{1}{1+x^2}$ $\to$ 积分得 $x\ln(1+x^2) - 2x + 2\arctan x + C$。

---

### 例 2（带参数）

求 $\displaystyle\int_0^{\sqrt{3}} \ln(3+x^2)dx$

**解**：套变形一公式（$a^2=3$）：

$$\Big[x\ln(3+x^2) - 2x + 2\sqrt{3}\arctan\frac{x}{\sqrt{3}}\Big]_0^{\sqrt{3}} = \sqrt{3}\ln 6 - 2\sqrt{3} + \frac{\pi\sqrt{3}}{3}$$

---

### 例 3（偶函数简化）

求 $\displaystyle I = \int_{-1}^{1} \ln(1+x^2)dx$

**解**：$\ln(1+x^2)$ 是偶函数，$I = 2\int_0^1 \ln(1+x^2)dx = 2\big(\ln 2 - 2 + \tfrac{\pi}{2}\big) = 2\ln 2 - 4 + \pi$。

---

### 例 4（直接分部 vs 凑微分）

求 $\displaystyle\int x\ln(1+x^2)dx$

**法一（凑微分）**：$\frac{1}{2}\int \ln(1+x^2)d(1+x^2) \to$ 结果。

**法二（分部）**：$u = \ln(1+x^2)$，$dv = xdx$，拆 $\dfrac{x^3}{1+x^2} = x - \dfrac{x}{1+x^2}$，结果一致。

---

## 易错点

1. **分部后忘记 $\arctan$**：$\displaystyle\int\frac{dx}{1+x^2} = \arctan x$，不是 $\arcsin x$ 也不是 $\ln(1+x^2)$。
2. **假分式直接积分**：$\dfrac{x^2}{1+x^2}$ 不可直接套 $\arctan$，必须先拆 $1 - \dfrac{1}{1+x^2}$。
3. **漏掉系数 2**：$du = \dfrac{2x}{1+x^2}dx$，第二项积分的系数是 $2$，不是 $1$。
4. **导数符号**：$\dfrac{d}{dx}\ln(1+x^2) = \dfrac{2x}{1+x^2}$，分子是 $2x$ 不是 $x$。
5. **$\dfrac{\ln(1+x^2)}{x}$ 当寻常积分硬算**：非初等，不可用分部直接得出初等形式。
